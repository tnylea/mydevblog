Flexbox simplifies layout tasks.

```css
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```

### Key properties:
- `justify-content`
- `align-items`
- `flex-direction`
- `flex-wrap`

Practice by recreating common layouts!
