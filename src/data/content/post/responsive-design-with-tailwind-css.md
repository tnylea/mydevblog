Tailwind provides utility classes for breakpoints.

```html
<div class="text-center md:text-left lg:text-right">
  Responsive text alignment
</div>
```

### Common breakpoints:
- `sm` → 640px
- `md` → 768px
- `lg` → 1024px
- `xl` → 1280px

Combine utilities to build responsive UIs faster.
