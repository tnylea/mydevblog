TypeScript adds static typing to JavaScript.

```ts
function greet(name: string): string {
  return `Hello, ${name}`;
}
```

### Benefits:
- Catch errors early
- Improved IDE support
- Better documentation

Try adding it gradually using `// @ts-check` or by converting files step by step.
