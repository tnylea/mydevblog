Svelte compiles your components into efficient JavaScript at build time.

### Highlights:
- No virtual DOM
- Simpler reactivity
- Smaller bundle sizes

Get started:

```bash
npx degit sveltejs/template my-svelte-app
npm install
npm run dev
```
